Lab5：推理與批次評估
## 題目說明
目標：

- 重新載入 base model + Lab4 訓練出的 LoRA 權重。
- 在推理時使用與訓練相同的 chat template：
    - add_generation_prompt=True
- 對 test.jsonl 的每筆資料進行推理，實作簡單評估：
    - 判斷是否有禮貌（包含「您好」或「請」）
    - 判斷是否結構化（包含「1)」「步驟」等）
    - 檢查是否以中文為主
- 計算平均分數及錯誤類型統計，並儲存到檔案。


完成後你應該能夠：

- 用微調後的模型批次推理。
- 有一個初步評估流程（哪怕只是啟發式）。
