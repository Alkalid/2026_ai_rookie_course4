Lab3：資料清洗與小型 SFT 資料集建立
題目說明
目標：

- 生成一個小型客服對話資料集。

- 客服的回應要有特定格式:
```python
顧客您好:
感謝您聯繫我們關於{topic}的問題。針對您的詢問："{user_query}"，我們建議您按照以下步驟進行：
1. 步驟一....
2. 步驟二....
3. 步驟三....
如果您在操作過程中遇到任何困難，請隨時與我們聯繫。我們的客服團隊將竭誠為您提供協助。
祝您有美好的一天！
客服團隊 敬上
```


- 對每筆資料進行：
    - 繁簡轉換（簡 -> 繁）
    - 去掉不必要空白、統一格式
    - 簡易毒性過濾（髒話過濾）
    - 長度檢查（太短/太長的對話不要）
    - 近似/重複樣本去重
    - 切分為 train/val/test 並輸出為 JSONL 檔。


完成後你應該能夠：

有一份乾淨、結構一致的 SFT 訓練資料，後續可直接拿來微調。
清楚了解常見的資料清洗步驟。
